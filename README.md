# 📚 Sistema de Gestão Escolar - Exercício de Funções em PHP

Este projeto foi desenvolvido como um exercício para praticar **funções em PHP**, dividido em quatro níveis de complexidade.  
O objetivo é demonstrar diferentes formas de criação e utilização de funções no PHP, incluindo funções com e sem parâmetros e com ou sem retorno.

---

# 🧑‍💻 Tecnologias Utilizadas

- **PHP**
- **HTML5**
- **CSS3**

---

# 📁 Estrutura do Projeto

```
htdocs
 └── Exerfunc
      ├── funcionalidade.php
      └── README.md
```

---

# ▶️ Como Executar o Projeto

1. Instale um servidor local como **XAMPP** ou **WAMP**.
2. Copie a pasta **Exerfunc** para dentro da pasta **htdocs**.
3. Inicie o **Apache** no painel do XAMPP.
4. Abra o navegador e acesse:

```
http://localhost/Exerfunc/funcionalidade.php
```

---

# ⚙️ Funcionalidades do Sistema

## 🔹 Nível 1 — Sem Retorno e Sem Parâmetro

Funções que executam tarefas diretas sem receber ou retornar valores.

- Exibir cabeçalho do sistema
- Mostrar log de inicialização
- Criar divisor visual
- Exibir rodapé com copyright

---

## 🔹 Nível 2 — Sem Retorno e Com Parâmetros

Funções que recebem dados como entrada e exibem o resultado diretamente na tela.

- Saudação personalizada
- Cálculo da área de um triângulo
- Formatação de valores em moeda (Real)
- Lista de compras utilizando **array** e **foreach**

---

## 🔹 Nível 3 — Com Retorno e Sem Parâmetro

Funções que retornam informações internas do sistema.

- Versão do sistema
- Gerador de cupom promocional
- Verificação de status de conexão com banco
- Fuso horário configurado no servidor

---

## 🔹 Nível 4 — Com Retorno e Com Parâmetros

Funções que recebem dados e retornam resultados após processamento.

- Cálculo de idade baseado no ano de nascimento
- Calculadora de desconto
- Validação de e-mail
- Conversão de temperatura (Celsius para Fahrenheit)

---

# 🎯 Objetivo do Projeto

Praticar conceitos fundamentais da programação em **PHP**, incluindo:

- Criação de funções
- Uso de parâmetros
- Retorno de valores
- Estruturas condicionais
- Manipulação de arrays
- Utilização de funções nativas do PHP
